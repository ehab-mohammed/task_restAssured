fake store api test automation

tools 
  -Java 
  -rest-assured to write automation code
  -testng    to using annotations
  -hamcrest  to easy to assert by using code like equal()
  -extentreports to generate file ExtentReport.html
  
project structure
-src/test/java/packges             
inclue three packges (baseAPI - listeners - task)

-testng.xml  
file to run tests and generat report

-pom.xml                        maven setup tools


how to run test script
-from code download zip file and extract manually
or 
-git clone [https://](https://github.com/ehab-mohammed/task_restAssured.git)
-download maven dependencies
-open the testng.xml file 
-right click "run testng"
